# Agri Grocer Data Analysis

## 📌 Description
This project focuses on analyzing data from an Agri Grocer dataset to extract meaningful insights and trends. The goal is to understand sales patterns, product performance, and customer behavior.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- Matplotlib
- Microsoft Excel

## 📁 Project Files
- app.py – Main analysis script that performs data analysis and generates insights  
- preload.py – Handles data loading and preprocessing  
- agri_grocer.db – Dataset used for analysis  

## ⚙️ How It Works
- The dataset is loaded and cleaned using `preload.py`  
- The cleaned data is then analyzed using `app.py`  
- Insights are generated based on the analysis  

## ▶️ How to Run
1. Install required libraries:
   pip install pandas matplotlib

2. Run the preprocessing file:
   python preload.py

3. Run the main analysis file:
   python app.py
